Это мой небольшой проект под названием «Календарь» Я пока только учусь, но хотел бы найти работу, связанную с написанием кода. Это домашнее задание с курсов.

Описание задачи

Задача создать календарь, в котором есть возможность делать записи на определенный день. Взаимодействие с пользователем должна осуществляться через консоль. Должна быть реализована возможность просмотра календаря, просмотра записей на определенный день, добавление записей (например, "в 14:00 стоматолог"). Реализовать возможность сохранения календаря в файл и восстановление из файла. Этот проект должен быть выполнен также с учетом принципов SOLID’а.

Реализация

Вся работы ситсемы будет разделена на несколько больших модулей:

  • Сохранение календаря
  • Вывод в консоль
  • Управляющий модуль

Сохранение календаря

Сохранение всех записей в календаре будет осуществяться с помощью базы данных на основе PostgreSQL. Модуль SQLAlchemy поможет общаться с базой данных

Вывод в консоль

Выводиться данные в консоль будут в виде таблицы содержащей данные о текущем дне недели, месяце, годе. Ниже будет идти шапка из дней недели, а строчки табицы будут заполнены днями текущего месяца. У пользователя будет возможность создать запись, посмотреть запись, изменить запись, сохранить запись, удалить запись (записи), а также выйти из "календаря"

Управляющий модуль
